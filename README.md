## Как установить
Скачайте и разархвириуйте код на своем компьютере.
Python3 должен быть уже установлен. Затем используйте pip (или pip3) для установки зависимостей:
```
pip install -r requirements.txt
```
## Как запустить
Создайте своего бота и получите Телеграм токен,создайте файл .env и напишите в нем следующее:
```
TOKEN='Токен вашего Телеграм-Бота' 

ADMIN_ID= Телеграм-ID админа бота
```
Чтобы запустить программу, напишите в терминал:
```
python main.py
```
